# PDP Visual Studio v12

Plataforma de generación automática de contenido visual para redes sociales.

**Input:** Keyvisual + PDP mensual + Cliente → **Output:** Keywords, Prompts IA, Piezas SVG editables.

## Cómo usar

1. Abrí el sitio
2. Cargá el Keyvisual del cliente (colores, mood, estilo)
3. Cargá el PDP (posteos del mes)
4. Click en **⚡ Generar todo**
5. Obtené: keywords creativos, prompts para Firefly/Gemini/DALL·E, y 10 piezas SVG

## Features

- **Keywords creativos** — 5 estructuras de búsqueda para bancos de imágenes
- **Prompts narrativos** — Prosa real, no sopa de tags (Guía v2)
- **10 piezas SVG** editables por posteo
- **✦ Mejorar con IA** — Refina keywords, prompts y diseños con Claude o Gemini
- **Selector Claude / Gemini** — Elige tu IA preferida
- **Perfiles de cliente** — Guarda configuraciones por marca
- **Canva** — Descarga SVG y abre Canva para editar
- **100% navegador** — No requiere instalación

## IA integrada

| API | Costo | Para qué |
|-----|-------|----------|
| Gemini (Google) | Gratis | Keywords, prompts, diseños |
| Claude (Anthropic) | ~$0.01/uso | Backup, mayor calidad |

Click en **🔑 Keys** para configurar tu API key.
